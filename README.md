# http3-rs

An async HTTP/3 implementation.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

**http3-rs** is a generic [HTTP/3][spec] implementation over pluggable QUIC transports and an [h3](https://github.com/hyperium/h3) fork focused on Client HTTP/3 support.

[spec]: https://www.rfc-editor.org/rfc/rfc9114
